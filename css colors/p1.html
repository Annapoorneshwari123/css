DEFINING CSS COLOR VALUES

colors in css most often specifices in the following formats:

  • color keyword -like green ,red ,blue etc.
  • Hexadecimal(2^8=256)value -like #000000,#ffffff and etc.
  • RGB(red,green,blue)value -like rgb(255,0,0),rgb(255,255,255),rgb(0,0,0) etc.
  • RGBA(red,green,blue,alpha)value -like rgb(255,0,0,0.5),rgb(255,255,255,0.5),rgb(0,0,0,0.5) etc.
  • HSL(hue,saturation,lightness) value
  • HSLA(hue,saturation,lightness,alpha) value
