css selectors

we have some css selectors to style the html elements

  • universal selectors(*)
  • element selectors (tag name)
  • class name selectors (class name using .)
  • Id selectors (using id attributes #)
  • attribute selectors (using any attributes inside the elements)
  • attribute with value selectors (attribute with values)
